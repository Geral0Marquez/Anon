git status: me proporciona informacion sobre el estado del repositorio. Rama actual,listado de comits, archivo en estado "untraked",archivo en staging area git add -A: me permite enviar todos los archivos en estado "untraked"hacia el staging area git add "nombreDelArchivo": envia SOLAMENTE el archivo indicado al staging area git commit -m "mensaje de commit": genera un nuevo commit con los archivos que se encontraban en el staging area,se asocia un mensaje que describe los cambios realizados con ese commit
